Creating GLSL hardware shaders learning path

The topics in this section provide guidelines for converting your GLSL shaders to use the .ogsfx format. The .ogsfx is a wrapper format that allows you to:

  • Combine various shader stages (such as the vertex shader, geometry shader and pixel shader) into one single effects file
  • Load your shaders via the glslShader.mll plug-in in Maya and visualize them in Viewport 2.0
