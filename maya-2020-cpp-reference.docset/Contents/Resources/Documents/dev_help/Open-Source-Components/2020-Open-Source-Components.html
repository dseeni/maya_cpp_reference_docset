Maya 2020 Open Source Components

This document provides the versions of the open source components used in Maya 2020.

Note: The following components have changed between Maya 2019.2 and Maya 2020.

  • IlmBase
  • libXML2
  • Glew
  • OpenImageIO
  • OpenJpeg
  • Ptex
  • PySide
  • Qt
  • TBB
