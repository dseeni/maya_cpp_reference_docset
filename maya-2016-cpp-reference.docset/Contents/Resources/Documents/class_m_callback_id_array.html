#include <MCallbackIdArray.h>

Class Description

Array of MCallbackId data type.

This class implements an array of MCallbackId. Common convenience functions are available, and the implementation is compatible with the internal Maya implementation so that it can be passed efficiently between plugins and internal maya data structures.

Examples:
cgFx/cgfxShaderNode.h , dagMessageCmd/dagMessageCmd.cpp , deletedMsgCmd/deletedMsgCmd.cpp , fileIOMsgCmd/fileIOMsgCmd.cpp , gpuCache/gpuCacheSubSceneOverride.h , lockEvent/lockEvent.cpp , nodeMessageCmd/nodeMessageCmd.cpp , polyMessageCmd/polyMessageCmd.cpp , testCameraSetCmd/testCameraSetCmd.cpp , and undoRedoMsgCmd/undoRedoMsgCmd.cpp .
